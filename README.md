# DA-PA-Checker

DA/PA Checker, verilen bir alan adı için Domain Authority, Page Authority ve benzeri metrikleri sorgulayan basit bir Chrome uzantısıdır.

## Ne Yapar

- kullanıcıdan hedef domain alır
- RapidAPI üzerinden metrik verisini çeker
- sonucu tablo halinde gösterir

## Teknolojiler

- JavaScript
- HTML
- Chrome Extension API

## Notlar

- Çalıştırmak için `popup.js` içindeki RapidAPI anahtarının doldurulması gerekir
